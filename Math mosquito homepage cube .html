Derivatives: The instentaneous rate of change

How to find the derivative of a function

Let's say there is a function. F(x)=x^2. To find the derivative of this we would do: [F(x+0.000000001)-F(x)]/0.00000000001. Let's do an example.

  1. First we write the formula: [F(x+0.000000001)-F(x)]/0.00000000001
  2. Next we write down the actual function: [(x+0.000000001)^2]-[x^2]/0.000000001
  3. Next we use one of the derivative tricks listed below: [2(x+0.000000001)]-[2(x)]/0.000000000001
  4. Finally we start to factor out numbers.

How to find the derivative of a function

Let's say there is a function. F(x)=x^2. To find the derivative of this we would do: [F(x+0.000000001)-F(x)]/0.00000000001. Let's do an example.

  1. First we write the formula: [F(x+0.000000001)-F(x)]/0.00000000001
  2. Next we use one of the derivative tricks listed below. F'(x)=2*x
  3. In this example there is not, but if need be factor out numbers.

The derivative of a function can be written as F'(x)=(Derivative)

Some tricks
  1. F(x)= x^n. F'(x)=n*[x^(n-1)]
  2. F(x)= x. F'(x)=x
  3. F(x)= x/n. F'(x)= x*(n^-1)
  4. F(x)= n/x. F'(x)= n*(x^-1)
